```
Definition of ready:

* Issue heeft wireframe (als mogelijk)
* Userstory en acceptatiecriteria moeten aanwezig zijn voor elke issue (als ... wil ik ... zodat)
* Iemand aan de issue gekoppeld
* Weight aan de issue gekoppeld
```

```
Definition of done:

* Overeenkomst tussen de groep
* Geen bugs/errors
* Gereviewed door PO
* Voldoet aan acceptatiecriteria
* Alle code is geformat met het `./vendor/bin/pint` commando.
* Push naar andere branch en vervolgens mergen naar Main

Probeer je zo veel mogelijk aan de [PSR-12](https://www.php-fig.org/psr/psr-12/) coding style guide te houden.
```
